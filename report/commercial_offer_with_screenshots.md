# Коммерческое предложение
## Проект: AI Task Manager для фрилансеров и агентств

## 1. Описание проекта

### 1.1. Цель проекта
Создать AI-инструмент, который автоматически приоритизирует рабочие задачи фрилансера, формирует выполнимый план на день/неделю и помогает снижать риск срыва дедлайнов по клиентским проектам.

### 1.2. Используемые технологии
- Python 3.10+
- Streamlit (веб-интерфейс)
- LangChain + OpenAI API (LLM-анализ задач)
- dotenv (`.env` для конфигурации)
- CSV/Markdown/ICS-экспорт для отчётности и календарей

### 1.3. Среда развертывания
- Локальный запуск на ПК исполнителя/клиента
- Возможность развертывания на облачном сервере (по согласованию)

## 2. Основные функции решения

### 2.1. Функциональные возможности
- Прием списка задач в структурированном текстовом формате.
- Анализ срочности, важности и ценности клиента.
- Расчет приоритета и сортировка задач.
- Формирование плана `today / tomorrow / this_week`.
- Генерация рекомендаций по рабочему фокусу.
- Выгрузка результата в `CSV`, `Markdown` и `.ics` для импорта в календарь.

### 2.2. Ключевые технические детали
- LLM возвращает строго структурированный JSON.
- Валидируется обязательная структура: `tasks`, `plan`, `recommendations`.
- Приоритизация основана на формуле с весами:
  `priority_score = 0.45 * deadline_score + 0.35 * importance_score + 0.20 * client_value_score`.

## 3. Этапы реализации проекта

1. Сбор и формализация требований к входным задачам.
2. Проектирование UX и интерфейса Streamlit.
3. Настройка промптов и LLM-цепочки.
4. Реализация логики приоритизации и генерации плана.
5. Добавление экспортов (`CSV/MD/ICS`) и проверка сценариев.
6. Подготовка демонстрации и проектной документации.

## 5. Ценность для клиента

- Сокращение времени на ежедневное планирование.
- Повышение фокуса на задачах с максимальной бизнес-ценностью.
- Снижение операционного стресса и риска перегруза.
- Прозрачная отчётность по задачам для клиента, наставника или команды.

## 6. Сроки, стоимость и условия

### 6.1. Сроки
Предварительный срок реализации MVP: **5-10 рабочих дней**.

### 6.2. Стоимость
Предварительная стоимость MVP-реализации: **от 35 000 руб.**  
Точная смета формируется после согласования детального ТЗ.

### 6.3. Условия
- Возможна работа по договору (самозанятый/ИП).
- Оплата по этапам: предоплата + финальный платеж после демонстрации результата.
- Поддержка и развитие проекта обсуждаются отдельным соглашением.
