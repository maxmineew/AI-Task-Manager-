# Коммерческое предложение
## Проект: ИИ-менеджер задач для фрилансеров

## 1. Краткое описание

ИИ-система для фрилансеров, которая анализирует входящие задачи, определяет приоритеты и формирует реалистичный план работы.  
Решение снижает хаос в ежедневном планировании и помогает быстрее закрывать задачи с максимальной ценностью.

## 2. Проблема клиента

Фрилансер одновременно ведет несколько проектов и сталкивается с:
- конфликтом дедлайнов;
- ручной, субъективной расстановкой приоритетов;
- перегрузкой и потерей фокуса;
- риском просрочек по ключевым клиентам.

## 3. Предлагаемое решение

Веб-приложение на `Streamlit + LangChain`, которое:
- принимает список задач и контекст;
- рассчитывает приоритеты по формуле с весами;
- формирует план `today/tomorrow/this_week`;
- дает рекомендации по организации рабочего процесса;
- экспортирует результат в `.csv` и `.md`.

## 4. Ценность для бизнеса

- Сокращение времени на планирование: до 30-60 минут в день.
- Рост дисциплины по дедлайнам и снижение просрочек.
- Фокус на задачах, влияющих на доход и удержание клиентов.
- Прозрачный отчет по плану для личного контроля или наставника.

## 5. Функциональные возможности

1. Анализ и структурирование задач.
2. Приоритизация по формуле:
   `priority_score = 0.45*deadline_score + 0.35*importance_score + 0.20*client_value_score`.
3. План выполнения задач по временным горизонтам.
4. Генерация рекомендаций по эффективности.
5. Экспорт результатов в отчеты.

## 6. Архитектура

Интерфейс (Streamlit)  
-> Логика обработки (Python, LangChain)  
-> LLM API  
-> Результат (структурированный список, план, рекомендации, экспорт)

## 7. Этапы реализации

1. Сбор требований и формата входных данных.
2. Разработка интерфейса.
3. Интеграция LLM и промптов.
4. Реализация приоритизации и планирования.
5. Тестирование на сценариях.
6. Подготовка документации и демо.
